# Jornadas

## Consumidor

Objetivo: encontrar comida boa perto dele por preco muito baixo.

Fluxo:

```txt
entrar no site/app
-> informar localizacao
-> ver achados proximos
-> aplicar filtros
-> abrir loja
-> escolher Achado Quaz
-> pagar
-> receber codigo de retirada
-> retirar no horario
-> avaliar
-> acompanhar impacto
```

Telas:

- onboarding.
- login/cadastro.
- home com achados.
- busca e filtros.
- detalhe da loja.
- detalhe do achado.
- checkout.
- pedido confirmado.
- meus pedidos.
- perfil.
- suporte.
- impacto pessoal.

## Parceiro

Objetivo: transformar excedentes em receita e reduzir desperdicio.

Fluxo:

```txt
preencher cadastro
-> aguardar aprovacao
-> configurar loja
-> criar Achado Quaz
-> receber pedidos pagos
-> preparar retirada
-> validar codigo
-> acompanhar vendas e avaliacoes
```

Telas:

- login parceiro.
- dashboard.
- minha loja.
- criar/editar achado.
- pedidos.
- financeiro.
- avaliacoes.
- suporte.
