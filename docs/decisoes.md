# Decisoes Do Projeto

## Decidido

Marca:

- Nome: Quaz di Graca em contexto tecnico.
- Nome em materiais finais: Quáz di Graça.
- Slogan: comida boa, preco quaz, desperdicio nao.
- Oferta padrao: Achado Quaz.
- Dominio principal: quazdigraca.com.br.
- Dominio secundario: quazdigraca.com, redirecionado para o dominio principal.

Estrategia:

- comecar web-first.
- app nativo depois da validacao.
- manter custo baixo.
- nao copiar codigo, marca, textos ou layout de concorrentes.

Tecnologia:

- GitHub privado.
- monorepo.
- Next.js.
- Vercel.
- Cloudflare.
- Supabase.
- Mercado Pago.
- React Native + Expo no futuro.

## A Confirmar

Gateway final:

- Mercado Pago.
- Pagar.me.
- outro.

Modelo financeiro:

- taxa por pedido.
- mensalidade para parceiro.
- combinacao dos dois.

Regras comerciais:

- percentual da plataforma.
- regras de cancelamento.
- regras de reembolso.
- tempo de reserva de estoque.
- cidades do piloto.

## Riscos

- pouca oferta de parceiros no inicio.
- baixa confianca do consumidor em oferta surpresa.
- qualidade inconsistente dos itens.
- cancelamentos por estoque indisponivel.
- custos de suporte se regras nao forem claras.

## Mitigacoes

- piloto controlado com poucos parceiros.
- aprovacao manual de parceiros.
- fotos e descricoes honestas.
- regras claras de retirada.
- suporte simples e rastreavel.
- metricas de cancelamento e avaliacao por loja.
